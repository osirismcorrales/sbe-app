# SBE App 📱

**SBE App** es una aplicación móvil moderna desarrollada con **React Native** y **Expo**, diseñada para facilitar la gestión de servicios estudiantiles y de bienestar. Permite a los usuarios realizar reservas, gestionar pagos, acceder a su credencial digital mediante código QR y mantenerse al día con los servicios disponibles.

---

## 🚀 Características Principales

- **🔑 Autenticación Segura**: Pantalla de inicio de sesión integrada.
- **📅 Reservas**: Módulo completo para agendar y confirmar turnos o espacios, con visualización de reservas activas.
- **💳 Pagos**: Gestión y seguimiento de pagos dentro de la aplicación.
- **🪪 Credencial Digital (Código QR)**: Acceso rápido a una credencial virtual interactiva mediante código QR para validación de identidad y acceso a servicios.
- **👤 Perfil de Usuario**: Edición de información del perfil directamente desde la aplicación.
- **🎨 Diseño Moderno**: Estilizado con Tailwind CSS para una interfaz fluida, rápida y atractiva.

---

## 🛠️ Tecnologías Utilizadas

- **React Native** & **TypeScript**
- **Expo** (Flujo de trabajo administrado / Managed Workflow)
- **Tailwind CSS** (para el estilizado de componentes)
- **React Navigation** (Navegación por pestañas y pilas de pantallas)

## 💻 Instalación y Configuración Local

Sigue estos pasos para ejecutar el proyecto en tu entorno local:

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/osirismcorrales/sbe-app.git
   cd sbe-app
   ```

2. **Instalar las dependencias:**
   ```bash
   npm install
   ```

3. **Iniciar el servidor de desarrollo de Expo:**
   ```bash
   npm run start
   # o bien:
   npx expo start
   ```

4. **Ejecutar en tu dispositivo:**
   - Escanea el código QR desde la aplicación **Expo Go** en tu dispositivo móvil (Android/iOS).
   - O presiona `a` para abrir en un emulador de Android, o `i` para un simulador de iOS.
